Setup Guide — Connect Your Google Sheet
1
Create your Google Sheet
Open Google Sheets and create a new spreadsheet. Add these exact column headers in row 1:
Student ID | Parent Phone | Student Name | Class | Session | Term | Mathematics | English | Basic Science | Social Studies | CRS | Civic Education | Computer Studies | Total | Average | Position | Attendance | Principal Comment | Teacher Comment
2
Get your Google Sheet ID
Copy the long ID from your sheet URL:
https://docs.google.com/spreadsheets/d/THIS_IS_YOUR_SHEET_ID/edit
Paste it as SHEET_ID in the CONFIG section of this file.
3
Enable Google Sheets API & get an API Key
Go to console.cloud.google.com → Create a project → Enable "Google Sheets API" → Go to Credentials → Create API Key.
Restrict the key to the Sheets API and your domain for security.
Paste it as API_KEY in the CONFIG section.
4
Share your Google Sheet
In Google Sheets: Share → Anyone with the link → Viewer. The API key can only read public/viewer-accessible sheets. Your data is still secure — only students with their own ID + parent phone can view their own record.
5
Optional: Use Google Apps Script for extra security
Instead of exposing an API key, deploy a Google Apps Script Web App that acts as a secure proxy. Create a new Apps Script in your Sheet (Extensions → Apps Script) and paste this code: 
function doPost(e) {
  var params = JSON.parse(e.postData.contents);
  var sid = params.studentId;
  var phone = params.parentPhone;
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = sheet.getDataRange().getValues();
  var result = [];
  for (var i = 1; i < data.length; i++) {
    if (String(data[i][0]).trim() == sid &&
        String(data[i][1]).trim() == phone) {
      result.push(data[i]);
    }
  }
  return ContentService
    .createTextOutput(JSON.stringify({ rows: result }))
    .setMimeType(ContentService.MimeType.JSON);
}
Deploy as a Web App (Execute as: Me, Who has access: Anyone). Copy the Web App URL and set CONFIG.APPS_SCRIPT_URL in this file to use it instead of the direct API.
