🔥 Firebase Setup Guide
1
Create Firebase Project
Go to console.firebase.google.com → Add project → name it → Create.
2
Register Web App & Copy Config
Click the </> icon → register app → copy firebaseConfig into FIREBASE_CONFIG at the top of this file's script.
3
Enable Firestore Database
Build → Firestore Database → Create → start in test mode → choose europe-west1 or nearest region.
4
Enable Firebase Storage
Build → Storage → Get started → test mode. Used for student passport photos.
5
Enable Email/Password Authentication
Build → Authentication → Get started → Email/Password → Enable. Then add all teacher accounts under the Users tab.
6
Create Staff Profiles in Firestore
For each Firebase Auth user, create a document in the staff collection (document ID = Firebase Auth UID):
staff/{uid}
  name:          "Mr. Chidi Nwachukwu"
  role:          "teacher"           // or "superadmin"
  assignedClass: "JSS 1A"            // null for superadmin
7
Data Structure
students/{studentId}
  name, parentPhone, classArm, gender, dob, photoURL
  results/{session-term}
    session, term, mathematics, english, …(by class)
    total, average, position, classSize, attendance
    teacherComment, principalComment
8
Firestore Security Rules
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Staff can read/write only students in their class
    match /students/{studentId} {
      allow read: if true;
      allow write: if request.auth != null
        && (get(/databases/$(database)/documents/staff/$(request.auth.uid)).data.role == 'superadmin'
           || resource.data.classArm.matches(
               get(/databases/$(database)/documents/staff/$(request.auth.uid)).data.assignedClass + '.*'));
      match /results/{resultId} {
        allow read: if true;
        allow write: if request.auth != null;
      }
    }
    match /staff/{uid} {
      allow read: if request.auth != null && request.auth.uid == uid;
      allow write: if request.auth != null
        && get(/databases/$(database)/documents/staff/$(request.auth.uid)).data.role == 'superadmin';
    }
  }
}
